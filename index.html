Q: Laptop Touchpad not working in Ubuntu / Linux

Try these commands in terminal

1) sudo -i
  (Enter your password then hit enter)
2) less /proc/bus/input/devices
  (check your touchpad vendor present or not, if present continue.....)
3) sudo rmmod i2c_hid
4) sudo modprobe i2c_hid
5) sudo apt-get update
6) sudo apt-get upgrade
